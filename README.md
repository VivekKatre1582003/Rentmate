# 🏠 RentMate – Smart Campus Rental Platform

**RentMate** is a full-stack rental platform for college campuses that allows students to list, rent, and manage shared items like books, electronics, bicycles, and more. The platform promotes sustainability and smart resource sharing within campus communities.

## ✨ Features

- 🔐 **User Authentication** – Register/login with Supabase Auth
- 📝 **List Items** – Upload item details and images
- 🛍️ **Rent Items** – Browse and request available items
- 📂 **Supabase Storage** – Upload and serve images
- 📊 **User Dashboard** – Manage your listings and rentals
- 🧠 **Spring Boot API** – Secure and robust backend logic
- 🌈 **Professional UI** – Based on dark theme (#2B2B2B) with vibrant accents:
  - Gold: `#FFD372`
  - Red-Orange: `#F15B42`
  - Pink: `#F49CC4`
- 💻 **Responsive Design** – Optimized for all screen sizes
